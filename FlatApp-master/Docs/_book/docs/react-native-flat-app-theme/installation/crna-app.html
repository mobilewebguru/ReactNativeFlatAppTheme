CRNA App

Flat App CRNA

Run the following commands on your terminal to setup Flat App CRNA Theme on your system.

Installation

  • Get the code

    • Opt #1. Download ZIP

      Not familiar with Git? Click here to download Flat App CRNA Theme. Extract the contents of ZIP file after downloading. Downloading ZIP file does not help you to sync with further updates of Flat App.

    • Opt #2. Clone

      To setup Flat App CRNA Theme on your system, and sync your app with constant updates, clone the repo. Click here to clone Flat App using GitStrap.

  • Install packages

    cd FlatApp/CRNA
yarn

  • Run on iOS

    • Opt #1:
      • Run npm start in your terminal
      • Scan the QR code in your Expo app
    • Opt #2:
      • Run npm run ios in your terminal
  • Run on Android
    • Opt #1:
      • Run npm start in your terminal
      • Scan the QR code in your Expo app
    • Opt #2:
      • Make sure you have an Android emulator installed and running
      • Run npm run ios in your terminal
